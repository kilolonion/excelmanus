# 颜色速查表

## 工具内置颜色名映射

format_cells 的颜色参数（font.color、fill.color、border.color）支持以下中文/英文名称，会自动转换为十六进制码：

### 基础色
| 名称 | 十六进制 | 示例用途 |
|------|---------|---------|
| 红/红色/red | FF0000 | 警告、负值标记 |
| 绿/绿色/green | 00B050 | 正值、通过状态 |
| 蓝/蓝色/blue | 0000FF | 链接、标题 |
| 黄/黄色/yellow | FFFF00 | 高亮、注意 |
| 白/白色/white | FFFFFF | 背景清除 |
| 黑/黑色/black | 000000 | 默认文字 |

### 常用色
| 名称 | 十六进制 | 示例用途 |
|------|---------|---------|
| 橙/橙色/orange | FFC000 | 警示、中等优先 |

| 粉/粉色/pink | FF69B4 | 装饰 |
| 棕/棕色/brown | 8B4513 | 标注 |
| 灰/灰色/gray | 808080 | 禁用、次要信息 |
| 青/青色/cyan | 00CED1 | 信息提示 |

## 常见美化场景样式组合

### 报表标题行
```json
{"font": {"bold": true, "size": 14, "color": "白色"}, "fill": {"color": "深蓝"}, "alignment": {"horizontal": "center"}}
```

### 表头行
```json
{"font": {"bold": true, "color": "白色"}, "fill": {"color": "天蓝"}, "alignment": {"horizontal": "center"}, "border": {"style": "thin"}}
```

### 数据行交替色
- 奇数行：无填充（默认白色）
- 偶数行：`{"fill": {"color": "浅灰"}}`

### 汇总行
```json
{"font": {"bold": true}, "border": {"top": {"style": "double"}, "bottom": {"style": "double"}}}
```

### 负值高亮
```json
{"font": {"color": "红色", "bold": true}}
```
