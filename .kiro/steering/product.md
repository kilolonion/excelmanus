# ExcelManus - 产品概述

ExcelManus 是一个基于大语言模型（LLM）的 Excel 智能代理框架。用户通过自然语言描述 Excel 任务，代理自动拆解意图并调用工具完成操作。

## 核心能力

- 自然语言驱动的 Excel 读写、分析、图表生成、格式化
- 支持数据透视表、公式填充、数据验证、条件格式等高级功能
- 基于 LangChain ReAct Agent 的多步推理与工具调用循环
- 默认接入阿里云通义千问（Qwen）大模型，可通过环境变量切换

## 目标用户

需要批量或智能化处理 Excel 文件的开发者和数据分析人员。
